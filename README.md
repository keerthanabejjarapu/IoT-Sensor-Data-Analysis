# IoT Sensor Data Analysis

## Project Overview
This project analyzes synthetic IoT sensor data using Python, NumPy, Pandas, and Matplotlib. The dataset was generated using NumPy and analyzed using Pandas. Various visualizations were created using Matplotlib.

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
The dataset contains 200 synthetic IoT sensor records with the following fields:
- Time
- Sensor_ID
- Temperature (°C)
- Humidity (%)
- Gas (ppm)

## Analysis Performed
- Generated synthetic sensor data
- Data inspection using Pandas
- Statistical analysis (mean, maximum, minimum, etc.)
- Data filtering
- Data visualization using line charts, bar charts, and histograms

- `sensor_data.xls`
- `IoT Sensor Data Analysis using Numpy, Pandas & Matplotlib.ipynb`
